# 임플라인치과 홈페이지 리뉴얼 PRD (Product Requirements Document)

## 1. 프로젝트 개요
- **프로젝트 명**: 임플라인치과의원 포천점 홈페이지 리뉴얼
- **목표**: 기존의 노후화된 홈페이지를 트렌디하고 고급스러운 디자인으로 전면 개편하여 병원의 브랜드 가치를 제고하고 환자들에게 신뢰감을 전달함.
- **주요 전략**: '한걸음더치과'의 미니멀하고 프리미엄한 레이아웃 및 디자인 시스템을 그대로 이식하여 임플라인치과만의 전문성을 강조.

## 2. 디자인 컨셉 및 방향성
- **참고 사이트**: 한걸음더치과 (https://onemorestepdental.com/)
- **핵심 무드**: 미니멀리즘, 깨끗함(Clean), 전문성(Professional), 프리미엄(Premium)
- **컬러 팔레트**: 
  - 메인 배경: Off-White (#F8F8F8 또는 화이트 기반의 미세한 톤)
  - 텍스트: 다크 그레이/차콜 (#333, #445)
  - 강조색: 임플라인치과 브랜드 컬러 (필요 시 최소한으로 사용)
- **타이포그래피**: 가독성이 높으면서도 고급스러운 명조체 또는 현대적인 고딕체 혼용 (예: 프리텐다드, 나눔명조 등)
- **레이아웃 특징**:
  - 넓은 여백(Margin) 활용을 통한 시각적 여유 확보
  - 고품질의 흑백/뮤트 톤의 의료진 및 장비 사진 활용
  - 스크롤에 따른 자연스러운 페이드인/업 애니메이션

## 3. 사이트 구조 (Sitemap)

| 대메뉴 | 소메뉴 (상세 페이지) | 레이아웃 참고 |
| :--- | :--- | :--- |
| **임플라인 소개** | 의료진 소개, 임플라인의 특별함, 병원 둘러보기 | 한걸음더 소개 섹션 |
| **임플란트** | 임플란트 안내 (전신질환 연계 등) | 한걸음더 임플란트 섹션 |
| **특화 진료** | 사랑니 발치, 수면 치료, 자연치아 살리기 | 각 특화 진료 섹션 |
| **보철/미백** | 심미보철치료, 치아미백, 턱관절 진료 | 관련 디자인 레이아웃 |
| **진료 안내** | 진료시간 안내, 오시는 길 (지도) | 하단 푸터 및 안내 섹션 |

## 4. 핵심 기능 요구사항
- **퀵 메뉴 (Floating Menu)**: 우측 상단/중단에 고정된 퀵 링크 아이콘
  - 네이버 예약
  - 네이버 블로그
  - 네이버 톡톡 상담
  - 카카오톡 플러스친구
  - 인스타그램
- **반응형 웹**: 데스크탑, 태블릿, 모바일 기기에 최적화된 레이아웃 제공
- **오시는 길**: 카카오 맵 API 연동 및 상세 주소/주차 안내
- **SNS 연동**: 인스타그램 피드 및 블로그 최신글 연동 (선택 사항)

## 5. 페이지별 상세 기획 (메인 페이지 기준)

### Hero Section (최상단)
- **좌측**: 임플라인치과만의 핵심 가치 문구 (예: "소신과 정직으로 환자의 편에서 진료하는 임플라인치과입니다.")
- **우측**: 미니멀한 그래픽 이미지 또는 '임플라인 —' 로고 강조 섹션
- **헤더**: 투명 배경에서 스크롤 시 반투명/화이트 배경으로 전환되는 상단 고정 바

### Medical Staff Section
- 대표 원장(허인범 원장)의 진료 철학과 고품질 흑백 프로필 사진 배치
- 텍스트: "1인 대표원장의 직접 진료로 처음부터 끝까지 책임집니다." 강조

### Treatment Specialization Section (Grid/Slide)
- 임플란트, 사랑니, 수면치료 등 주요 서비스를 사각형 프레임과 미니멀한 텍스트로 표현
- 한걸음더치과의 방식대로 한 섹션에 하나의 테마씩 스크롤하며 강조

### Location & Info Section
- 하단에 깔끔한 지도와 진료 시간, 사업자 정보 배치
- 카피라이트 및 로고 포함

## 6. 기술 요구사항
- **Framework**: Next.js 또는 Vite (빠른 로딩과 SEO 최적화)
- **Styling**: TailwindCSS 또는 Vanilla CSS (미니멀한 디자인 구현 용이)
- **SEO**: 각 페이지별 메타 태그, 검색 엔진 최적화
- **Performance**: 이미지 최적화 (WebP 활용) 및 Lottie/Framer Motion 기반의 가벼운 애니메이션

2. 프론트엔드 퍼블리싱 (메인 및 상세)
3. 기능 연동 및 테스트
4. 최종 배포
